Инструкция для Android TV

Vless - AndroidTV

  1. Скачиваем и сохраняем на флешку приложение v2rayNG
    Берем у бота ключи в виде QR-кода для XRay. Для этого нажмите в боте "Подписка" -> "QR Vless" и сохраняем его на флешку.
  2. Для удобства добавления QR-кода, установите файловый менеджер, например File Manager Plus.
  3. Подключите флешку к ТВ и установите приложение, которое вы сохранили ранее.
  4. Откройте установленное приложение и добавьте туда QR-код, который вы сохранили на флешку.
  5. После того как вы добавите QR-код, появится список серверов. Выберите любой из них и нажмите круглую кнопку для подключения.

Второе приложение: v2rayTun

  1. Устанавливаем v2raytun через Google Play
  2. Переходим в телеграмм через браузер
  3. Переходим в бот - меню - подписка - копируем ключ Vless (нажимаем одной кнопкой пульта по ссылке)
  4. Запускаем v2raytun - нажимаем «Управление» - Импорт из буфера обмена
  5. Ждем загрузки локаций - нажимаем на нужную нам локацию - нажимаем на кнопку «Подключение» - обязательно даем согласие
  6. Готово ✅
